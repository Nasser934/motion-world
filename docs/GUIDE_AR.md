# دليل motion-world

## ما تفعله المهارة

المهارة لا تصنع فيديوًا فقط. هي تبني خط إنتاج كامل:

1. تستلم وصف الحركة والمنصة المستهدفة.
2. تنشئ صور البداية والنهاية والمرجع عبر ChatGPT أو Codex أو أي مصدر.
3. ترسل الصور إلى Higgsfield أو مزود آخر.
4. تستلم الفيديو وتفحصه.
5. تعيد ترميزه إلى ملف سريع البحث أو تستخرج منه إطارات وSprite Atlas.
6. تنشئ Runtime يناسب iOS أو Android أو Flutter أو React Native أو Web.
7. تربط الحركة بقيمة من 0 إلى 1 قادمة من الوقت أو العداد أو السكرول أو السحب أو الحساس أو حالة التطبيق.

## مثال

جلسة مدتها 20 دقيقة:

```text
progress = elapsed / 20 minutes
```

هدف 10 خطوات:

```text
progress = completedSteps / 10
```

صفحة تمرير:

```text
progress = scrollOffset / scrollDistance
```

كلها تشغل نفس الحركة لأن الـRuntime لا يعرف معنى القيمة؛ هو يستقبل نسبة فقط.

## اختيار صيغة الحركة

- فيديو قصير سينمائي: MP4 قصير GOP يتحرك بالبحث داخل الزمن.
- حالة تحتاج دقة عند كل نسبة: إطارات WebP.
- عنصر صغير جدًا: Sprite Atlas.
- حركة بسيطة قابلة للتفكيك: طبقات Native.
- أجهزة ضعيفة أو Reduce Motion: 3 إلى 11 صور أساسية مع Crossfade.

## Higgsfield

المهارة تنشئ Plan أولًا وتفحص نموذج Higgsfield قبل التشغيل. بعض النماذج لا تدعم صورة نهاية أو دقة أو مدة أو صوت بنفس الخيارات، لذلك لا تُضاف الخيارات عشوائيًا.

## سُكون

سُكون مثال فقط داخل `references/examples/sukun-oasis.motion-project.json`. لا توجد أي ملفات من المهارة داخل مستودع سُكون، ولا يعتمد قلب المهارة عليه.
